# SRS Buddy

Telegram-бот для изучения слов с интервальным повторением: создавайте и импортируйте колоды, добавляйте карточки, проходите ежедневные сессии и получайте напоминания.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
